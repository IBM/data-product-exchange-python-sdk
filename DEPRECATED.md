# ⚠️ REPOSITORY DEPRECATED

This repository has been **deprecated** and is **no longer maintained**.

## New Repository

All development has moved to: **[IBM Data Intelligence SDK](https://github.com/IBM/data-intelligence-sdk)**

## Migration Guide

### 1. Uninstall the Old Package

```bash
pip uninstall dph-services
```

### 2. Install the New Package

```bash
pip install ibm-data-intelligence-sdk
```

### 3. Update Your Code

The new SDK may have different import paths and API changes. Please refer to the [new repository's documentation](https://github.com/IBM/data-intelligence-sdk) for:

- Updated API documentation
- Migration examples
- Breaking changes
- New features

### 4. Update Dependencies

If you have `dph-services` in your `requirements.txt` or `pyproject.toml`, replace it with:

```
ibm-data-intelligence-sdk
```
